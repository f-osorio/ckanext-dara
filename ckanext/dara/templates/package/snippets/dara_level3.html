Metadata Level 3
Interrelate all kinds of scientific output

By integrating authority files a linkage across various data stocks is enabled (for instance it is possible to link persons to their affiliations, to other publications, their institutions or to their respective fields of research). Also future reuse of these (meta)data by semantic web technologies is enabled. 25 metadata fields with additional subfields are required. In addition all IDs for the different authority files must be submitted. Not all information of the so called data lifecycle can be included.
